# X Source Review for Muse Image

twitterapi.io is a discovery tool, not a publication pipeline. Raw responses remain in ignored local storage.

## Search Matrix

Rotate exact model queries with production intent: `Muse Image prompt`, `Muse Image edit`, `Muse Image reference`, `Muse Image typography`, `Muse Image portrait`, `Muse Image carousel`, and creator-thread replies.

## Admission Gates

Every community entry requires:

- explicit Meta Muse Image evidence;
- a canonical creator source and attribution;
- complete prompt text or an honestly labeled reconstruction;
- result media attached to the source;
- one primary workflow;
- prompt-location metadata;
- no duplicate source, normalized text, media URL, or perceptually matching result.

## Ranking and Publication

Prefer cases that expose agentic research, precise editing, reference composition, readable layout, sequential consistency, or deliberate art direction. Reject launch announcements, generic model mentions, reposts, hidden prompts, and unrelated-model results. Published records retain source IDs, prompt provenance, engagement snapshot, model evidence, and collection date without exposing credentials or raw search caches.
