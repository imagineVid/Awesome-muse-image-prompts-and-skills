# Muse Image Workflow Taxonomy

Classify each case by the production decision that makes it reusable. Every community prompt receives exactly one primary workflow.

## Workflow Families

### Agentic Research & Factual Visuals
Use when search, calculation, code, or current information is essential to the image. The source should make the factual request explicit.

### Precision Editing & Scene Preservation
Use for a localized change that must preserve composition, lighting, identity, geometry, or unaffected objects.

### Multi-Reference Composition & Identity
Use when several references must be combined while a person, product, garment, or design system remains recognizable.

### Typography, Posters & Structured Layouts
Use when readable text, hierarchy, spacing, and layout rules determine success.

### Sequential Art & Social Formats
Use for carousels, panels, story sequences, or other multi-frame formats that require cross-frame consistency.

### Portraits, Texture & Art Direction
Use when likeness, material rendering, lighting, portrait language, or a deliberate art direction is the main production problem.

## Decision Rule
Choose the one instruction that would make the workflow fundamentally different if removed. Do not assign every plausible category; a clear primary purpose is more useful than tag inflation.
