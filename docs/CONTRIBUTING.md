# Contributing a Muse Image Case

This repository publishes evidence-backed workflows, not model mentions or prompt-shaped marketing copy.

## Evidence Bundle

Submit:

- the complete public prompt, or a reconstruction clearly labeled as such;
- the canonical creator post rather than an aggregator or repost;
- one or more result images attached to that source;
- creator name, profile, publication date, and source language;
- explicit evidence that the result used Meta Muse Image;
- required reference inputs and one primary workflow category.

If the prompt appears in a reply, quoted post, or ALT text, identify its exact location. Editorial normalization may improve grammar or complete clearly stated variables, but it must never be presented as a verbatim source prompt.

## Review

Maintainers verify provenance, model evidence, result media, reusable detail, category fit, and textual or visual duplicates. Availability claims must follow Meta's current first-party material. Engagement can aid discovery but cannot replace evidence.

## Rights and Removal

Attribution remains attached to every public case. Creators and rights holders may open an issue for correction or removal; verified requests receive priority.
